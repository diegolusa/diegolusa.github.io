    Fontes

    A escolha de uma fonte é crucial para o sucesso de uma interface web, visto que está diretamente ligada à experiência de leitura do usuário (que deve ser agradável, naturalmente).

    As propriedades de estilo que o CSS3 nos fornece para manipularmos fontes são font-family, font-style, font-weight, font-size, e font-variant. Na sequência iremos abordar cada uma delas:

    font-family¶

    Especifica a família de fontes que o navegador deve utilizar para renderizar o texto. A propriedade aceita vários valores separados por vírgula, indicando opções de fontes quando não for possível utilizar a anterior. Normalmente finalizamos especificando uma das cinco famílias genérias de fonte disponíveis: serif, sans-serif, monospace, coursive e fantasy 1 2 3.

    A família genérica deve ser utilizada para contornar a possibilidade do usuário não ter disponível em seu navegador as outras famílias listadas. Logo, nestes casos, o navegador conseguirá preservar o padrão tipográfico com outra fonte semelhante disponível 1 2 3.

    Cada família de fontes genérica tem características tipográficas particulares. Vejamos quais são elas.

    • serif: Cada glifo (símbolo) de caractere apresenta traços, terminações, de acabamento.

    Exemplo de texto utilizando fonte serifada.

    • sans-serif: Os glifos não apresentam traços de acabamento.

    Exemplo de texto utilizando fonte sem serifas.

    • monospaced: Cada glifo ocupa o mesmo espaço (mesmo tamanho). Assemelha-se com a forma de escrita das antigas máquinas de escrever.

    Exemplo de texto utilizando fonte monoespaçada.

    • cursive: Os glifos assemelham-se a escrita manual.

    Exemplo de texto utilizando fonte cursiva.

    • fantasy: São fontes artísticas.

    Exemplo de texto utilizando fonte artística.

    font-style¶

    Propriedade que permite especificar a apresentação da face dos glifos, se normais, em itálico ou oblíquos 2 3. Os possíveis valores são:

    • normal

    Normal (padrão).

    • italic

    Em itálico.

    • oblique

    Oblíquo (quase igual ao itálico).

    font-size¶

    Permite especificar o tamanho da fonte, a partir de medidas relativas, como percentual, em, rem, etc ou medidas absolutas como pixels. Para saber mais sobre unidades de medida, acesse nosso conteúdo sobre o tema 2 3.

    16px - 2rem - 2em

    font-weight¶

    Define o peso (intensidade de negrito) da fonte. Os valores podem ser normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900 e inherit. Os valores numéricos indicam uma graduação de negrito. O valor 400 equivale a normal enquanto que o valor 700 é equivalente a bold. Os valores bolder e lighter aplicam-se em relação ao peso herdado do ancestral, enquanto que normal e bold aplicam-se em absoluto ao texto (sem relação com o valor do ancestral) 2 3.

    Valor 200 - Valor 400 - Valor 600 - Valor 900

    font-variant¶

    Indica se o texto deve ser exibido ou não considerando apenas caracteres maiúsculos. Os valores possíveis são small-caps e normal (padrão) 2 3.
